🚀 Vercel 무료 배포 가이드 - AI 단어 암기장

📋 준비사항
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

✅ GitHub 계정 (무료)
✅ Vercel 계정 (무료)
✅ OpenAI API 키


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
1단계: GitHub 저장소 생성
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. GitHub (https://github.com) 접속 및 로그인
   (계정이 없으면 회원가입)

2. 우측 상단 "+" 클릭 → "New repository" 선택

3. 저장소 정보 입력:
   ┌─────────────────────────────────────┐
   │ Repository name: vocabulary-app     │
   │ Description: AI 단어 암기장         │
   │ Public 선택                          │
   │ ✅ Add a README file (체크 해제)   │
   └─────────────────────────────────────┘

4. "Create repository" 클릭

5. 생성된 저장소 페이지에서 URL 복사
   예: https://github.com/사용자명/vocabulary-app.git


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
2단계: GitHub에 코드 업로드
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

PowerShell을 열고 다음 명령어 실행:

cd "C:\Users\Mr Kim\Desktop\Vocabulary"

git remote add origin https://github.com/사용자명/vocabulary-app.git

git branch -M main

git push -u origin main

⚠️ GitHub 로그인 창이 뜨면 로그인하세요.


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
3단계: Vercel 계정 생성 및 연결
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. Vercel (https://vercel.com) 접속

2. "Sign Up" 클릭

3. "Continue with GitHub" 선택
   → GitHub 계정으로 로그인

4. Vercel이 GitHub 접근 권한 요청
   → "Authorize Vercel" 클릭


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
4단계: 프로젝트 배포
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. Vercel 대시보드에서 "Add New" → "Project" 클릭

2. GitHub 저장소 목록에서 "vocabulary-app" 찾기
   (안 보이면 "Adjust GitHub App Permissions" 클릭)

3. "Import" 클릭

4. 설정 화면:
   ┌─────────────────────────────────────┐
   │ Project Name: vocabulary-app        │
   │ Framework Preset: Other             │
   │ Root Directory: ./                  │
   │ Build Command: (비워두기)           │
   │ Output Directory: (비워두기)        │
   └─────────────────────────────────────┘

5. "Environment Variables" 섹션 펼치기

6. 환경변수 추가:
   ┌─────────────────────────────────────┐
   │ Key: OPENAI_API_KEY                 │
   │ Value: sk-proj-실제API키입력        │
   └─────────────────────────────────────┘
   "Add" 클릭

7. "Deploy" 버튼 클릭 🚀


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
5단계: 배포 완료 및 접속
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. 배포 진행 (1-2분 소요)
   "Building..." → "Deploying..." → "Ready" ✅

2. 배포 완료 후 URL 확인
   예: https://vocabulary-app-xxx.vercel.app

3. "Visit" 클릭하여 접속 테스트

4. 스마트폰, 태블릿 등 모든 기기에서 접속 가능!

🔧 코드 수정 시 재배포 방법
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

코드를 수정한 후:

cd "C:\Users\Mr Kim\Desktop\Vocabulary"
git add .
git commit -m "업데이트 내용"
git push

→ Vercel이 자동으로 재배포! (1-2분)

🎯 사용자 지정 도메인 연결 (선택사항)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Vercel 프로젝트 → Settings → Domains
→ 원하는 도메인 연결 (무료)

❓ 문제 해결
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

❌ "Function exceeded time limit" 오류
   → OpenAI API 응답 지연, 잠시 후 재시도

❌ "API 키 오류"
   → Vercel 환경변수 OPENAI_API_KEY 확인

❌ "404 Not Found"
   → vercel.json 파일 확인
   → public/index.html 파일 존재 확인

❌ GitHub push 오류
   → git remote -v 로 URL 확인
   → GitHub 로그인 상태 확인
