🎓 AI 단어 암기장 - 설치 및 실행 가이드

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
✅ 1단계: Node.js 설치 확인
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

명령 프롬프트(CMD)를 열고 다음 명령어를 입력하세요:

    node --version

✅ 버전이 표시되면 Node.js가 설치되어 있습니다.
❌ 오류가 나면 https://nodejs.org 에서 다운로드하세요.


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
✅ 2단계: 의존성 패키지 설치
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

현재 폴더에서 명령 프롬프트를 열고 다음을 입력:

    npm install

(필요한 패키지: express, cors, node-fetch 자동 설치)


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
✅ 3단계: OpenAI API 키 설정
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. https://platform.openai.com/api-keys 접속
2. API 키 생성 및 복사
3. 아래 명령어 중 하나로 환경변수 설정:

   방법 1 (PowerShell - 추천):
   ─────────────────────────────────────────
   $env:OPENAI_API_KEY="sk-xxxxx"

   방법 2 (CMD):
   ─────────────────────────────────────────
   set OPENAI_API_KEY=sk-xxxxx

   방법 3 (.env 파일 사용):
   ─────────────────────────────────────────
   1) .env.example을 복사하여 .env로 이름 변경
   2) .env 파일에 API 키 입력:
      OPENAI_API_KEY=sk-xxxxx

   3) server.js에 다음 코드 추가 (맨 위):
      require('dotenv').config();

   4) dotenv 패키지 설치:
      npm install dotenv


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
✅ 4단계: 서버 실행
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

    npm start

또는

    node server.js


서버가 실행되면 다음과 같이 표시됩니다:

🔧 문제 해결
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

❌ "포트가 이미 사용 중" 오류
   → 다른 프로그램이 3000번 포트 사용 중
   → server.js에서 PORT를 3001로 변경

❌ "API 키가 설정되지 않음" 오류
   → 3단계 다시 확인
   → API 키를 올바르게 입력했는지 확인

❌ "번역 오류" 또는 "AI 추천 오류"
   → 인터넷 연결 확인
   → API 키가 유효한지 확인
   → API 크레딧 잔액 확인
